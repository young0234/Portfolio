2025년 7월 16일 레파지토리에 추가됨

1. 프로젝트 개요

  • | 정보 전달 중심 영역의 디자인 및 퍼블리싱 전체.
  • | 기여도 : 100%(메인(네비, 배너, 영화 슬라이드, 푸터), 서브(무비즈, 마이페이지, 로그인, 404페이지))
  • | 작업기간 : 2025.07.10. - 2025.07.16. (5h - 6h, weekdays only)

2. 결과물

2.1. 메인페이지

2.1.1. 배너 및 메인 비주얼 섹션

  • 상단에 네비게이션 바를 제작하여 서브 페이지로의 이동을 편리하게 구성했습니다.
  • 배너 안의 ‘트레일러’ 버튼을 클릭하면 트레일러 영상이 재생됩니다.

2.1.2. 카드 슬라이드 섹션

  • 각 제목에 맞춰 카드들이 정렬되어 표시됩니다.
  • 카드에 마우스를 올리면 간단한 정보가 나타납니다.

2.2. 서브페이지

2.2.1. 영화 검색 및 전체 페이지

  • 드롭다운 메뉴를 활용해 원하는 기준으로 정보를 정렬할 수 있습니다.
  • 검색 기능을 통해 원하는 영화를 빠르게 찾을 수 있습니다.
  • 검색 결과가 없을 경우, ‘검색 결과가 없습니다’라는 문구와 함께 하단에 인기 영화 목록을 표시합니다.

2.2.2. 영화 상세 페이지

  • 상세 페이지에는 배너, 출연 배우 정보, 관람 후기가 나열되어 있음.
  • 출연 배우 목록은 스와이퍼 기능을 통해 더 많은 배우들을 확인할 수 있음.

4.2.3. 마이 페이지

  • 좌측에는 개인정보가 나열되어 있으며, 그 아래에 여러 기능 버튼들과 로그아웃 버튼이 배치되어 있음.
  • 우측에는 찜한 영화와 평점을 남긴 영화 목록이 스와이퍼로 구성되어 있어 쉽게 확인할 수 있음.

4.2.4. 로그인 페이지

  • 전체 레이아웃은 깔끔하게 정렬하되, 선택한 색상으로 포인트를 주어 시각적 강조를 더함.
  • 입력 내용 없이 로그인 버튼을 클릭하면 경고창이 나타나도록 설정함.

4.2.5. 404 페이지

  • 영화 페이지에 어울리도록 팝콘 모양 요소를 추가해 시각적인 매력을 높임.
  • 상단에 ‘홈으로 돌아가기’ 버튼을 배치하여 원래 화면으로 쉽게 이동할 수 있도록 함.

5. 성과 및 회고

  • | TMDB API를 활용해 풍부한 데이터를 이용할 수 있어 매우 유익했습니다.
  • | 리액트로 여러 페이지를 작업하는 것이 처음이라 힘들었지만, 완성해서 뿌듯합니다.
  • | 정렬과 검색 기능 구현이 쉽지 않았지만, 꼭 필요한 기능이라 완성 후 만족도가 큽니다.
  • | 앞으로도 꾸준히 배우고 성장하는 웹퍼블리셔로 발전해 나가겠습니다.
