날씨 앱 (Weather)

Single

only PC

2025년 7월 14일 레파지토리에 추가됨

1. 프로젝트 개요

  • | Weather API를 활용한 React 기반 반응형 날씨 웹사이트 작업.
  • | 기여도 : 100%(메인페이지)
  • | 작업기간 : 2025.07.08. - 2025.07.14. (5h - 6h, weekdays only)

2. 결과물

2.1. 메인페이지

4.1.1. 전체적 섹션

  • 하단 버튼을 클릭하면 휴대폰 화면의 내용과 배경이 변경됨.
  • 우측 버튼을 클릭하면 클릭 효과가 나타나 눌렀음을 알 수 있음.

4.1.2. 모달 창 섹션

  • 상세한 정보는 모달 창을 통해 확인할 수 있음.
  • 추후에는 모달이 아닌 별도의 서브 페이지로 이동하도록 변경할 예정임.

5. 성과 및 회고

  • | 리액트로 처음 작업한 결과물이라 아쉬운 점도 있지만, 다음 프로젝트에서는 더 나은 결과물을 기대할 수 있을 것 같습니다.
  • | 현재는 메인 페이지만 구현되어 있지만, 향후에는 서브 페이지가 포함된 작업물을 만들고 싶습니다.
  • | Weather API를 활용해 다양한 정보를 쉽게 가져올 수 있어 매우 유용했습니다.
  • | 앞으로도 꾸준히 배우고 성장하는 웹퍼블리셔로 발전해 나가겠습니다.
